__init__¶

This file lists the objects to be used in a normal execution of pyvsim. This way, the user does not have to know the internal distribution of classes (which may look somewhat arbitrary, e.g. Volume is in Primitives, not in Toolbox)

The idea is to implement a factory, so that all component registration and deregistration is done automatically in the future.
